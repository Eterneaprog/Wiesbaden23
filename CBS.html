About CASD and our data

  • 10 years of data -> 20 000 exports / 350 000 files
  • Manual controls for the majority of projects
  • Automatic controls for some specific projects

Current state of the project

  • Open several type of files and extract the texts / tables / images
  • Get the label of an export
  • Generate variables on the metadata of the export / the file
  • Train a model on this data and call manual control when risk is high

Current development

  • Train the model file by file (dropping export by export approach)
  • Analyse content of a table / text / image
  • Use form data, user data, projects data, sources used
  • Continuous automated training

The future

  • Possibly an LLM for text
  • Possibly a Deel Learning model for images
  • A graphical user interface for controls
